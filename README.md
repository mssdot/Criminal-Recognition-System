# Criminal-Recognition-System

Developing a Desktop Application for Criminal Detection using deep learning technologies in various media like videos, images, and live camera feeds, with a focus on implementing swift and accurate identification. The application will incorporate email notifications to alert authorities upon detecting criminal activities, enhancing proactive law enforcement measures
